Goal,

For this evaluation engine, we need to develop an api that create an object based step by step array, in which it can take in the parameters of the problem and analyze the general cases, the special cases, and everything in between to give the student understanding that 0 x 5 or 1 x 5 is easier than 5 x 5 for an inherent reason.
